# Task System

- Source of truth: `Backlog.md CLI`
- Backing metadata root: `backlog/`
- Read interface: `backlog task <id> --plain`
- Write interface: `backlog task create|edit ...`
- Plan location: task `Implementation Plan`
- Notes location: task `Implementation Notes`
- Completion location: task status `Done` plus `Final Summary`
- Rule: all code changes must have a task
- Rule: create a task if none exists
- Rule: do not edit task files directly
